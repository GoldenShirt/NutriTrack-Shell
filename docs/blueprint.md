# **App Name**: NutriTrack

## Core Features:

- Voice/Text Input: Enable voice and text input for logging meals using device's voice services.
- AI Meal Analysis: Use AI to classify and structure meals by date and provide nutrition insights.
- AI Nutrition Chat: Allows users to chat with an AI tool, focused on improving user nutrition.
- Daily Dashboard: Displays a simple visual summary of daily nutritional intake (calories, protein, carbs, fats).
- Context saving: Saves all meal context data, tells the ai the context for recent entries from the past 7 days; retains summary-style memory of older interactions for long-term history.

## Style Guidelines:

- Primary color: HSL 210, 70%, 50% (approx. #3399E6). This vibrant blue conveys health and clarity.
- Background color: HSL 210, 20%, 95% (approx. #F0F8FF). This provides a soft, clean backdrop.
- Accent color: HSL 180, 70%, 40% (approx. #33CCCC). This complementary color highlights key actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text
- Use clear, simple icons representing different food groups and nutrients.
- Responsive layout with a focus on clean lines and intuitive navigation.
